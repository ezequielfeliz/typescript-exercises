# TypeScript Exercises

A personal collection of TypeScript exercises created for learning and practicing the language. This repository is a work in progress and grows as I work through new concepts and challenges.

## About

This repo holds small, self-contained exercises written in TypeScript. Each exercise focuses on a specific concept, helping to build a solid understanding of the language step by step.

## Topics covered

- Basic types and type annotations
- Interfaces and type aliases
- Functions and generics
- Classes and object-oriented patterns
- Union, intersection, and utility types
- Asynchronous code with promises and async/await

## Getting started

To run the exercises locally, make sure you have Node.js and TypeScript installed.

```bash
# Clone the repository
git clone https://github.com/ezequielfeliz/typescript-exercises.git

# Move into the project folder
cd typescript-exercises

# Install dependencies (if a package.json is present)
npm install

# Compile a TypeScript file
tsc filename.ts

# Run the compiled output
node filename.js
```

## Project structure

Each exercise lives in its own file or folder, named after the concept it covers. This makes it easy to find and revisit specific topics.
